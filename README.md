# Student Management System (Java)

A console-based Student Management System developed using Java.
This project demonstrates Object-Oriented Programming concepts
and basic data handling.

## Features
- Create student records
- Update student details
- Search students
- Delete records
- Display all students

## Technologies Used
- Java
- OOP Concepts
- Console-based I/O

## How to Run
1. Compile: javac Main.java
2. Run: java Main
